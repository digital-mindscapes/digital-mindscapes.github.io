How to Read a Bivariate Map

The 3x3 Grid

A bivariate map shows two different variables simultaneously on one map. We use a 3x3 color grid to represent the combination of values:

  • Horizontal Axis: Represented by Metric X. Moving from left to right shows increasing values for Metric X.
  • Vertical Axis: Represented by Metric Y. Moving from bottom to top shows increasing values for Metric Y.

Interpreting Colors

  • Light Grey (Bottom-Left): Low values for both metrics.
  • Deep Blue (Top-Left): High Y metric, Low X metric.
  • Deep Pink (Bottom-Right): Low Y metric, High X metric.
  • Dark Purple (Top-Right): High values for both metrics (High-High).

Why use this?

It helps identify spatial correlations. For example, if you see many dark purple counties, it means those areas struggle with both issues (e.g., high unemployment and high obesity).
