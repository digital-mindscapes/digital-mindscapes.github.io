Clustering Methodology & Interpretation

What is Clustering?

Clustering is an unsupervised machine learning technique that groups similar data points together. In this analysis, we use it to identify counties with similar health and socioeconomic profiles.

Algorithms Explained

1. K-Means Clustering

Partitioning data into K distinct, non-overlapping clusters. It works by minimizing the variance within each cluster.

  • Best for: Finding spherical, well-separated groups.
  • The Elbow Method: Helps determine the optimal K by looking for the point where adding more clusters provides diminishing returns in variance reduction.

2. DBSCAN (Density-Based Spatial Clustering of Applications with Noise)

Groups together points that are close to each other based on a distance measurement (Epsilon) and a minimum number of points (Min Points).

  • Best for: Finding clusters of arbitrary shapes and identifying outliers (Noise).
  • Noise: Points that do not belong to any cluster are labeled as -1.

3. Hierarchical Clustering (Agglomerative)

Builds a hierarchy of clusters by starting with each point as its own cluster and merging the closest pairs until the desired number of clusters is reached.

  • Linkage Criteria: Determines how "distance" is measured between groups (e.g., Ward's method minimizes variance, Single linkage uses the closest points).

Interpreting Visualizations

  • Scatter Plot: A 2D projection (PCA or t-SNE) of high-dimensional data. Points close together have similar feature values.
  • Cluster Summary: Shows the size and distribution of each identified group.
  • County Assignments: Use the right sidebar to search for specific counties and see their cluster membership.
