About


An Interactive Public Health & Socioeconomic Data Visualization Platform

The primary purpose of this project is to provide a comprehensive, interactive platform for visualizing public health and socioeconomic data across the United States. By combining data from various reliable sources, it allows researchers, policymakers, and the general public to explore spatial patterns, identify disparities, and understand the complex relationships between demographic factors, health outcomes, and socioeconomic indicators at both state and county levels.

Data Source

The Digital Mindscapes platform aggregates and visualizes data from two primary, highly reliable national sources to provide a comprehensive view of public health and socio-economic factors across the United States.

PLACES (Population Level Analysis and Community Estimates)

PLACES is a free, interactive web tool that provides chronic disease and other health-related data for small areas across the United States. Originating in 2015 as a partnership between the Centers for Disease Control and Prevention (CDC), the Robert Wood Johnson Foundation, and the CDC Foundation, it offers high-quality estimates. This data is critical to help state and local health officials, policymakers, and communities identify local health problems, prioritize resources where disparities exist, and develop targeted health initiatives.

ACS (American Community Survey)

The American Community Survey (ACS) is an ongoing, annual demographics survey program conducted by the U.S. Census Bureau. It provides vital information on a yearly basis about the nation and its people, generating data that helps inform how federal funds are distributed. It regularly gathers detailed insights ranging from jobs, occupations, and educational attainment, to income, housing, and structural variables. Public officials, planners, and researchers use this information to assess community needs and provide vital context to public health data.

On the results page, click Download Table Data (CSV). This gives raw files for use in R, Python, or Excel.

2. Access Data via the Census API (Programmatic Access)

For automated access or custom scripts, use the U.S. Census API base: api.census.gov/data.html

1
Get an API Key

Sign up for a free API Key at census.gov. This is required for many API calls.

2
Choose Your Dataset

Pick a popular dataset like ACS 1-year/5-year, Decennial Census, or Population Estimates.

3
Construct Your Query

Use the base format:
https://api.census.gov/data/<YEAR>/<DATASET>?get=<VARIABLES>&for=<GEOGRAPHY>

4
Example Query (All Counties)

To fetch total population (B01003_001E) for all counties:
https://api.census.gov/data/2024/acs/acs5?get=NAME,B01003_001E&for=county:*

5
Process & Save

Use libraries like R: censusapi or Python: requests to fetch and convert results into CSV/JSON.

Data Preparation

The accuracy and reliability of the Digital Mindscapes platform depend on meticulous data preparation. Our data pipeline involves several key steps to ensure seamlessly integration between diverse data sources.

Key Steps in Preparation

  • 1. Data Cleaning: Raw files from sources like CDC PLACES and the US Census ACS are carefully inspected. Missing values (N/A) and erroneous entries are flagged. Where appropriate, missing numerical datasets are handled through logical interpolation or omission based on statistical best practices.
  • 2. Standardization: Datasets from different sources often use varied naming conventions or geographic IDs. FIPS codes (Federal Information Processing Standards) are rigorously verified and formatted to a consistent 5-digit string to serve as the unified geographic key uniting all health and socio-economic variables.
  • 3. Structural Alignment: Variables are mapped out to follow a consistent naming schema within structured dictionaries. This ensures that a metric like "Average Income" aligns perfectly across states and counties.

Data Processing

Transforming gigabytes of raw prepared data into a visually coherent and responsive dashboard requires efficient programmatic data processing.

Key Processing Methodologies

  • 1. Programmatic Aggregation (Python): Utilizing tools like Pandas, large files containing detailed block-level or zip code-level metrics are rolled up to County and State levels. Averages are calculated factoring in population weighting to reflect accurate true representative values.
  • 2. JSON Transformation: The vast tabulated data is ultimately transformed into lightweight JSON object arrays. This ensures the web application loads quickly on the client-side while still holding thousands of data points locally.
  • 3. Regional Groupings: Automated scripts dynamically append geographic identifiers like "Census Region", "Division", and "RUCC Group" onto data objects based on predetermined mapping lists, allowing dynamic filtering out of the box.
  • 4. Dynamic Extremes: Max/Min bounds for gradients and heatmap legends are calculated algorithmically based directly upon the dataset provided, rather than being hardcoded, allowing automated scalability to future variables.

Main Dashboard

The Main Dashboard serves as the primary entry point to the Digital Mindscapes platform. It presents a comprehensive, interactive map of the United States, providing a macroscopic view of various socioeconomic and health indicators at the state level.

Key Features

  • Interactive State Map: Hover over states to instantly view specific metric values via dynamic tooltips.
  • Metric Selection: Effortlessly switch between dozens of health outcomes (e.g., Diabetes, Asthma) and socio-economic factors (e.g., Income, Education) through the organized sidebar.
  • Choropleth Visuals: The map automatically updates its color gradient (heat map) to visually represent the intensity or prevalence of the selected metric across the country.
  • Responsive Design: Fully optimized for both desktop and mobile devices, ensuring accessibility everywhere.

States Comparison

The States Comparison page is designed for detailed, customized evaluation of specific states. It allows users to actively select multiple states and perform side-by-side analysis using versatile chart and table views.

Key Features

  • Custom Selection: Click on individual states on the map to add them to your comparison group. The map highlights selected states while greying out unselected ones.
  • Versatile Charting: View the selected states' data through dynamic charts. Toggle smoothly between Horizontal Bar Charts, Vertical Bar Charts, and Treemaps.
  • Detailed Table View: Switch to a tabular layout to see the raw numbers for your selected states side-by-side.
  • Region Grouping: Use the "Region" toggle to quickly group or color-code states based on their geographic region (e.g., Midwest, Northeast).
  • Heatmap Mode: Toggle the map's heatmap coloring on or off without losing your current state selections.

Counties Comparison

The Counties Comparison page provides granular, local-level insights. It functions similarly to the state comparison tool but drills down to over 3,000 individual US counties.

Key Features

  • County Selection: Interactively click counties across the map to build your custom comparison dataset.
  • Rich Visualizations: Analyze your selected counties using clear Horizontal or Vertical Bar Charts. For larger selections, expand the chart to a full-screen modal for better readability.
  • State Grouping: When viewing the data table, use the "Group by State" toggle to automatically organize selected counties under their respective state headers.
  • RUCC Filtering: Filter the entire map view by Rural-Urban Continuum Codes (e.g., "Metropolitan", "Nonmetro") to observe discrepancies between urban and rural localized regions.
  • Region Visualization: Color-code your selected counties by geographical region on the charts to spot broader geographic trends.

Urban-Rural Comparison

The Urban-Rural Comparison page focuses specifically on highlighting the disparities and patterns between metropolitan and non-metropolitan areas across the country. It utilizes the USDA's Rural-Urban Continuum Codes (RUCC) to categorize counties.

Key Features

  • RUCC Focused Filtering: Apply strict filters to show only counties belonging to specific RUCC classifications right on the interactive map.
  • RUCC Grouping Toggle: Organize selected counties in charts and tables specifically by their Urban/Rural status to draw immediate visual comparisons between population densities.
  • Flexible Chart Modes: Switch between robust Horizontal and Vertical Bar Charts to display the data in the clearest way possible depending on your selection size.
  • Seamless Integration: Retains all the powerful map interactive features, tooltip data, and tabular layouts found in the Counties Comparison page.

State Multi-Metric Comparison

The State Multi-Metric Comparison tool allows for complex analysis by examining the correlation between two distinct variables. It answers questions like: "Is there a relationship between education level and heart disease prevalence?"

Key Features

  • Dual-Metric Selection: Select a 'Primary Metric' mapped to the X-axis and a 'Secondary Metric' mapped to the Y-axis.
  • Scatter Plot Visualization: View the interaction of two variables across all 50 states via an interactive scatter plot. States are plotted as circles—hover over them to read exact values for both metrics.
  • Region Filtering: Use quick-filter buttons (Midwest, Northeast, South, West) to isolate specific geographical regions within the scatter plot.
  • Dual Data Table: Switch to table view to see a clean, sorted list of states with their respective values for both the primary and secondary variables side-by-side.
  • Color-Coded Regions: Scatter plot points are intelligently color-coded based on their US Census Region, making grouping patterns instantly recognizable.
