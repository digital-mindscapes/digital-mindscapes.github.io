Statistical Methodology

The Pearson Correlation Coefficient (r) measures the linear correlation between two variables, returning a value between -1 (perfect negative correlation) and 1 (perfect positive correlation).

r = Σ((xi - x̄)(yi - ȳ)) / √(Σ(xi - x̄)² Σ(yi - ȳ)²)

Interpretation Guidelines

  • |r| ≥ 0.6: Strong correlation (distinct, highly concentrated linear trend)
  • 0.3 ≤ |r| < 0.6: Moderate correlation (visible relationship, clear direction)
  • |r| < 0.3: Weak correlation (highly scattered relationship)

Technical Notes

  • Computations are handled strictly client-side via the simple-statistics JavaScript module.
  • Missing or null records for either axis variable drop that row entirely (Pairwise Deletion).
  • The trendline inside the scatterplots represents an Ordinary Least Squares (OLS) simple linear regression (y = mx + b), rendering only if n > 5 and |r| > 0.05.
