Methodology & Interpretation

What is PCA?

Principal Component Analysis (PCA) is a technique used to reduce the dimensionality of large datasets. It transforms a large set of variables into a smaller one that still contains most of the information in the large set.

How to Interpret the Visualizations

1. Scree Plot & Cumulative Variance

These charts tell you how much "information" (variance) each component captures.

  • The Elbow Method: Look for the "elbow" or bend in the scree plot. Components before this bend are typically the most significant.
  • Rule of Thumb: Aim for a subset of components that captures at least 70-80% of the total cumulative variance.

2. 2D Biplot (The Map)

This plot displays both the observations (counties) as points and the variables as vectors (arrows).

  • Points: Counties that are close together have similar profiles across the selected variables.
  • Vectors (Arrows):
    • Direction: Vectors pointing in the same direction are positively correlated. Opposite directions indicate negative correlation.
    • Length: Longer arrows indicate that the variable has a stronger influence on the two displayed components.
    • Angles: A 90-degree angle between two arrows suggests those variables are not correlated.

3. Component Loadings (Weights)

This table shows the mathematical weight of each variable on the first three components.

  • Values range from -1 to 1.
  • A high absolute value (e.g., > 0.4 or < -0.4) indicates that the variable is a strong driver of that specific Principal Component.

ⓘ How to interpret these results

This table breaks down exactly how the Principal Component score for this specific county is mathematically derived. PCA abstracts your data into independent underlying dimensions by systematically weighting every variable you chose.

  • Component Weight (Loading): This represents how strongly this specific variable influences the selected Component (e.g. PC1). This weight is universal for the entire dataset—it does not change from county to county.
  • Standardized Value: Because health metrics run on completely different scales (e.g. raw populations vs. percentages), every single data point is converted into a Z-score. A Standardized Value of 1.5 literally means this specific county is exactly 1.5 standard deviations above the national average for that metric.
  • Score Contribution: This is the most important column! It is simply the (Weight × Standardized Value). It calculates the exact amount this variable mathematically shifted the county's final Component score. If you scan down this column, the variables with the largest absolute numbers (magnitudes furthest from zero) are the dominant driving reasons why this county is colored dark red or dark blue on the map!
Example: If the Component Weight for High Blood Pressure is 0.423, and a county has a shockingly high Standardized Value of 1.500 for High BP (1.5 standard deviations above the mean), its Score Contribution from High BP alone would be (0.423 × 1.500) = +0.635. That massive positive jump aggressively pulls the county's color toward Red on the map.
